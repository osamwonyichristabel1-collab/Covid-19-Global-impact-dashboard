# Covid-19-Global-impact-dashboard

## Tools used
- Tableau public[data visualisation and dashboarding]
- Excel[for initial data exploration]
- kaggle[dataset source]

## Project overview
An interactive data dashboard analyzing the global impact of covid-19 across
countries.This project explores case trends,death rates,recovery patterns and
vaccination progress to uncover meaningful insights from real world pandemic data.
Built entirely in tableau ,the dashboard enables users to interact with data dynamically.

## What i learned
- How to build interactive map visualisation.
- How to tell compelling data story throught visualisation.

## key insights discovered
- USA sustained the highest recorded cases.
- Despite highest casualties,USA led global vaccination deployment.

## Skills demonstrated
- Interactive dashboard development.
- Geographic map visualization.
